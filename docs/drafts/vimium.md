# Vimium 使用指南

## 常用键

- f click F open in this/new tab
- h j k l 左下上右
- gg top G bottom
- d down u up a lot
- ~~a+~~r R reload
- yy copy the url
- f2 switch the focus
- p P open copied url in this/new tab
- gu/U go up/root
- yi copy the image
- [[ / ]] go previous/next page
- gf/F next/main frame
- o / O open
- b / B search in book marks
- T search in tabs
- / find in this page
- n / N 上/下一处
- H L 后退 前进
- K/gt J/gT next/previous tab
- g0 / g$ first /last tab
- t new tab
- yt copy to new tab
- x X close/restore the tab



## others

- a-p toggle
- a-m mute
- << >> move
- ? help
